### What I Learned
- **Apt vs tarball tradeoff**: apt Go 1.22.2 match exactly dengan `reference/entropy-course/go.mod` (go 1.22). Tarball lebih fleksibel (bisa pilih 1.26.x) tapi risk mismatch. Pilih apt untuk presisi.
- **Binary path gotcha**: `apt install golang-1.22` install ke `/usr/lib/go-1.22/bin/go`, BUKAN `/usr/local/go/bin`. Docs default assume tarball path. Solution: `export PATH=/usr/lib/go-1.22/bin:$PATH`.
- **`vendor/` is a Go-reserved folder name**: Go detect folder `vendor/` sebagai Go's vendor mechanism dan complain kalau gak ada `modules.txt`. Pake nama `reference/` lebih akurat karena emang bukan Go vendor — ini cuma copy folder belajar.
- **Dep version drift**: gw pin `pgx/v5@v5.5.5` (match entropy-course), tapi dependency-nya (`golang.org/x/text v0.29.0`) udah naik minimum Go version ke 1.24. Solusi: terima `go.mod` declare `go 1.24.0` (toolkit dev, bukan production). Catatan untuk project lo yang production-grade: be more careful with dep pinning.
- **Vertical slice pattern (entropy-course)**: 13 commands, masing-masing slice 1:1 dengan 1 usecase. Tidak ada "shared usecase" antar command. Kunci hexagonal — biar setiap command testable independen.
- **Dependency rule yang ketat**: domain zero external imports, ports speak only domain types, adapters implement ports. Usecase constructor receives interfaces, NEVER concrete types.
- **Timestamps via Clock port**: entities ga punya `Clock` — usecase yang inject `now time.Time` ke entity methods. Ini bikin test deterministic dan domain pure.
- **13 decisions di roadmap**: ini bukan kerja random — Ko Stephen catat SEMUA keputusan penting. Mindset: "kenapa bukan alternatif lain?" adalah pertanyaan wajib sebelum pilih.
- **Compile-time port assertion**: pattern Go-idiomatic — `var _ core.CourseRepository = (*PostgresCourseRepository)(nil)`. Kalau lo lupa implement satu method, compile error langsung.
- **Convention over ceremony**: VOs stay VO kalau ada invariant. Title yang cuma butuh non-empty? Stay `string`. Wrapping dalam VO = ceremony tanpa payoff.
