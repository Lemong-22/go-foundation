# Journal Index

Weekly progress logs. Update tiap hari sebelum push.

| Week | Date Range | Phase | File | Status |
|---|---|---|---|---|
| 1 | 2026-06-11 → 2026-06-17 | Foundation (Go + Postgres) | `2026-WEEK-23.md` | 🟡 In progress |

Format: ISO week number (`YYYY-WW.md`).

## Template

Lihat `journal/TEMPLATE.md` untuk struktur entry harian.

## Cara Pakai Journal

- **Pagi sebelum mulai**: tulis target hari ini di section "Today's Goal"
- **Siang/sore sebelum push**: update "What I Did" + "What I Learned" + "Blockers" + "Tomorrow"
- **Akhir minggu**: tulis retrospective di section "Week Retrospective"
- **JANGAN skip journal** — ini single source of truth untuk progress lo, lebih reliable dari ingatan
